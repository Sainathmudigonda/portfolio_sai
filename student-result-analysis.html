🎓 Student Result Analysis

Objective: Analyze student performance and discover how study hours, sleep, attendance, and previous scores affect exam results.

🔍 Tools Used

  • Python (Pandas, NumPy, Matplotlib, Seaborn)
  • Scikit-learn for predictive modeling
  • Jupyter Notebook / VS Code

📈 Project Highlights

  • Explored correlations between study habits and results.
  • Built regression & classification models to predict performance.
  • Generated clean reports for dashboards or analytics tools.

📊 Key Insights

  • Higher study hours and attendance correlate with better results.
  • Proper sleep improves performance consistency.
